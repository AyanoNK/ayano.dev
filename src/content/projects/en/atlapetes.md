A fully developed Android mobile application to **recognize an endangered bird species** called "Atlapetes Blancae" using a Machine Learning image classification model.

## Key Features

- **ML Image Classification**: Real-time bird species recognition using PyTorch
- **Camera Integration**: CameraX for live camera feed and image capture
- **Field Research Tool**: Designed for use in rural areas of Antioquia, Colombia
- **Offline Capable**: Works in remote areas with limited connectivity

## Impact

Researchers are currently using this application to identify the endangered Atlapetes Blancae bird in the wild. The Humboldt Institute reserves the application usage rights, and special permission is required to download and use the application.

## Technical Highlights

The app uses a PyTorch-based image classification model trained on images of the Atlapetes Blancae species. The model runs on-device through Android's ML inference capabilities, allowing researchers to identify birds in real-time through the camera feed. The CameraX API provides reliable camera access across a wide range of Android devices.
