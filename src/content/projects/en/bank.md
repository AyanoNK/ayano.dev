An Android application designed to help users **budget their everyday spending** by automatically reading bank SMS messages and generating financial metrics.

## Key Features

- **Automatic SMS Parsing**: Reads and categorizes bank transaction messages
- **Monthly Expense Reports**: Generates detailed metrics from reported expenses
- **Material3 Design**: Built with Jetpack Compose and Material3 for a modern UI
- **Home Screen Widget**: Jetpack Glance widget for quick expense overview

## Technical Highlights

The application leverages Android's SMS permissions to parse incoming bank notifications, extract transaction amounts, and categorize them into spending categories. The data is stored locally and processed to generate monthly financial reports with visual charts.
