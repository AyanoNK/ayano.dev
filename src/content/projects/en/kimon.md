A web application designed to help **Japanese Kanji learners** practice reading and recognizing kanji through music lyrics.

## Key Features

- **Music-Based Learning**: Practice kanji recognition using real Japanese song lyrics
- **Interactive Interface**: Built with NextJS and React for a smooth learning experience
- **Progress Tracking**: Powered by Supabase for user data and learning progress
- **Responsive Design**: Works seamlessly across desktop and mobile devices

## Technical Highlights

The platform uses the Supabase backend for authentication and data storage, with a NextJS frontend deployed on Vercel. The learning algorithm presents kanji in context through song lyrics, making the learning process more engaging and memorable.
