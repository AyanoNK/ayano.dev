Una aplicación web diseñada para ayudar a los **estudiantes de Kanji japonés** a practicar la lectura y el reconocimiento de kanji a través de letras de música.

## Características Principales

- **Aprendizaje Basado en Música**: Practica el reconocimiento de kanji usando letras reales de canciones japonesas
- **Interfaz Interactiva**: Construida con NextJS y React para una experiencia de aprendizaje fluida
- **Seguimiento de Progreso**: Impulsado por Supabase para datos de usuario y progreso de aprendizaje
- **Diseño Responsivo**: Funciona perfectamente en dispositivos de escritorio y móviles

## Aspectos Técnicos

La plataforma utiliza Supabase como backend para autenticación y almacenamiento de datos, con un frontend en NextJS desplegado en Vercel. El algoritmo de aprendizaje presenta kanji en contexto a través de letras de canciones, haciendo el proceso de aprendizaje más atractivo y memorable.
